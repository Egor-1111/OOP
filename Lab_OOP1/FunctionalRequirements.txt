1) После создания приложения, приложение должно отображать меню с функциями (Нарисовать фигуру, переместить нарисованную фигуру, 
Удалить нарисованную фигуру, Сохранить комбинацию фигур в файле, Загрузить из файла сохраненные данные или пустой файл, заполнить другой фон
нарисованными фигурами, Отменить действие, повторить действие)
2)Нарисуйте фигуру - эта функция дает пользователю возможность выбрать одну из пяти фигур (круг, прямоугольный треугольник, треугольник с углом 90°,
прямоугольник, эллипс). Чтобы выбрать фигуру, пользователь должен ввести цифру в диапазоне [1..5] круг - 1, прямоугольный треугольник - 2, треугольник с углом 90° - 3,
прямоугольник - 4, эллипс - 5.
После выбора фигуры для каждого варианта программа выдает специальные параметры для каждой фигуры:
круг - радиус
прямоугольный треугольник - сторона
треугольника с 90 углами - две стороны (2 катета)
прямоугольник - две стороны
эллипса - два радиуса
После выбора специального параметра выбранная фигура будет нарисована и помещена на первый план игрового поля
3) Переместить нарисованную фигуру - эта опция доступна ТОЛЬКО при наличии фигур на игровом поле.
Выбрав одну из нарисованных фигур, пользователь переходит в специальный режим перемещения (каждая сторона (вверх, вниз, влево, вправо) имеет специальный номер [1..4]
для выхода из специального режима введите -1).
4) Удалить нарисованную фигуру - эта функция доступна, только если на доске есть какие-либо фигуры.
После выбора одной из нарисованных фигур (они описаны в диапазоне [0..количество фигур-1]), чтобы выбрать фигуру, введите цифру из указанного диапазона.
Остальные выбранные фигуры исчезнут
5) Сохранить комбинацию рисунков в файле - эта функция сохраняет все рисунки в файле storage.txt
6) Загрузить из файла - эта функция загружает данные из файла. Если файл пуст, рабочее пространство очистится
7) Заполнить другой фон из нарисованных фигур - после выбора одной из нарисованных фигур. Программа выдает 5 различных фоновых символов.После ввода 
фон цифры в диапазоне [1..5] изменится, и цифра выйдет на первый план.
8) Отменить действие - эта функция удаляет последнее действие. Если никаких действий не будет, программа предоставит пустое рабочее пространство, пользователь сможет использовать только 5
раз подряд эту функцию!
8) Повторить действие - эта функция возвращает последнюю удаленную функцию, если она была удалена. Только пять раз подряд.Если нет никаких действий, программа выдаст пустое рабочее пространство.
